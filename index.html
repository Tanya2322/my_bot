О проекте

Ассистент
это программа которая
помогает вам в вашей
повседневной жизни
помогает с делами
и конкретными задачами

я писал её на Python код получился большой более 300 строк

я сделал картинки самой работы бота они справа

Как это работает?

  1. Пользователь делает запрос

    Для этого создали удобный интерфейс
  2. Программа использует специальные библиотеки

    По запросу пользователя выбирается библиотека
  3. Запрашивает данные с нужного сервера

    Программа запрашивает данные
  4. Преобразует запрос в голос и озвучивает

    Для преобразования текста в голос программа
    использует библиотеку SpeakerPy
